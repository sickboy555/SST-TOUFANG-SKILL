# 高频问题作战卡

这份文档把 `何俊杰` 语料库里反复出现的高频问题压成一组“先判断什么、先别做什么、先怎么动”的作战卡。

## 1. 预算 / 出价 / 单价突然变了

### 更像是什么

- 可能是曝光池变了。
- 可能是学习节奏被你推快了。
- 也可能只是系统开始吃更贵但更准的人群了。

### 先看什么

1. 当前是不是测试期 / 冷启动期。
2. 最近是否明显加过预算、改过出价、换过目标、换过素材。
3. CPM、CTR、CVR、后端回收分别怎么变。
4. 版位和人群结构有没有变化。

### 当前最不建议做的事

- 看见单价变贵就立刻提价。
- 还没分清曝光质量和转化质量，就给计划判死刑。

### 如果是我来处理

1. 先分清是“曝光更散了”还是“转化用户更差了”。
2. 再看当前预算是否把系统推到了不适合的探索强度。
3. 最后决定是收预算、稳几天、拆计划，还是补新素材。

## 2. 平台后台和 MMP / 三方数据对不上

### 更像是什么

- 多数时候不是谁在偷量，而是口径没对齐。
- 也可能是 SAN / SKAN 机制差异、回传配置问题、时区问题。

### 先看什么

1. 平台和三方看的时间段是不是一样。
2. 时区是不是一样。
3. 归因窗口是 1 天、7 天，还是 click / view 混算。
4. 是 SAN 还是普通网络。
5. 回传事件、事件映射、deep link / referrer 配置有没有问题。

### 当前最不建议做的事

- 只看半天数据就说平台崩了。
- 没对齐口径就去大改投放动作。

### 如果是我来处理

1. 先对齐时间、窗口、时区。
2. 再分清平台口径和三方口径分别在统计什么。
3. 最后再判断是统计差、归因差，还是技术回传故障。

## 3. IAA / IAP / 混合变现该怎么投

### 更像是什么

- 先别想着“全都要”，先判断产品主引擎在哪。
- 真问题常常不在投放动作，而在回收周期、商业化设计和人群拆分。

### 先看什么

1. IAA 和 IAP 各自贡献占比。
2. 回收周期是 D0、D3、D7 还是更长。
3. 当前回传的是付费、广告、还是混合 ROAS。
4. 产品内有没有“潜在付费用户”的中间行为。

### 当前最不建议做的事

- 用一个目标硬吃两套变现逻辑。
- 只按 D0 ROAS 判断所有模式。

### 如果是我来处理

1. 先分产品更偏 IAA、IAP，还是双强。
2. 再分人群：付费人群、广告人群、潜在付费人群。
3. 最后决定是混投、拆投，还是混合回传但拆考核。

## 4. 素材还能不能继续扛

### 更像是什么

- 素材问题不只是“好不好看”，而是现在筛进来的还是不是你想要的人。
- 老素材跑不动，未必是投手懒，也可能是流量池和平台算法阶段变了。

### 先看什么

1. 素材当前承担的是起量任务、筛选任务，还是回收任务。
2. CTR、CVR、留存、后端回收、生命周期分别怎么变。
3. 不同国家、语言、平台、受众下是不是同样疲劳。
4. 老素材是否被跨账户、跨平台、跨目标复用太久。

### 当前最不建议做的事

- 因为它以前好用，就一直硬扛。
- 只看 CTR 说素材还行。

### 如果是我来处理

1. 先看这个素材筛进来的人还值不值钱。
2. 再判断它是该淘汰、做衍生，还是换个使用场景。
3. 有条件的话，把素材回收到素材库，补标签和历史表现。

## 5. 团队总定位不到真实问题

### 更像是什么

- 大概率不是单点能力差，而是问题拆解方式错了。
- 更常见的是管理层不看细节、数据体系不完整、考核把动作逼变形。

### 先看什么

1. 有没有清楚分前端、后端、口径、组织四层问题。
2. 管理层有没有定期俯身看数。
3. 数据链路、埋点、回传、归因有没有缺口。
4. 一线有没有试错空间，还是天天被结果倒逼。

### 当前最不建议做的事

- 一出问题就怪素材、怪账户、怪优化师。
- 上面只要结果，不让下面解释过程。

### 如果是我来处理

1. 先把问题按层拆开。
2. 再明确每一层谁负责、缺什么数据、缺什么动作。
3. 最后再谈是继续投、停计划、补系统，还是重做协同机制。

## 6. 述职 / 复盘怎么写

### 更像是什么

- 不是单纯汇报花了多少钱。
- 真正有价值的是“你比别人看懂了什么、你比过去进步了什么、你接下来打算怎么干”。

### 先写什么

1. 基础数据：消耗、效果、利润，按产品 / 渠道拆开。
2. 对比同行：行业水平、竞品判断、自己的优势和不足。
3. 对比过去：上半年 / 去年到现在的进步。
4. 后续计划：投放技术、产品节奏、行业趋势、具体动作。

### 当前最不建议做的事

- 全篇只有消耗和 revenue。
- 没有趋势判断，没有下一阶段计划。

### 如果是我来处理

1. 先讲结果。
2. 再讲对比和进步。
3. 最后落到趋势和计划，让汇报有“下一步”。
